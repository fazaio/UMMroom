Web Server
1. Install XAMPP
  • Download XAMPP sebagai web server, untuk mendowload silahkan klik link berikut (Download)
  • Pilih Bahasa
  • Pilih komponen, Silahkan dicentang semua karena kita akan membutuhkan semua komponen yang disediakan XAMPP
  • Setelah itu tunggu beberapa menit sampai proses selesai
2. Jalankan Apache & Mysql
  • Klik tombol Start untuk menjalanakan Apache, Mysql, Filezilla
Extract zip (ummroom.zip)
  • Download Sourcecode UMMroom disini (Download)
  • Extract File pada folder XAMPP
Database
1. Buka phpmyadmin & import SQL
  • Login Phpmyadmin
  • Klik Import > browse > pilih sql yg akan diimport
  • Tunggu sampai proses selesai
Testing
1. Buka Localhost pada Web browser
  • Kunjungi http://localhost/UMMroom
  • Pilih menu Sensor
2. Selamat Mencoba
