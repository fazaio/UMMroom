UMMroom
  • adalah sebuah platform aplikasi online untuk memantau keterpakaian ruang kelas dengan sistem IoT menggunakan sensor PIR dan Nodemcu, kemudian setelah data didapatkan dari maka data akan disimpan secara realtime pada database MYSQL.
Tujuan
  • Visualisasi data dari sensor ke Web Server
  • Mengembangkan IoT dengan teknologi Web server Sebagai pengelolahan database dan visualisasi database
Manfaat
  • Untuk memudahkan memantau ruang kelas secara Realtime menggunakan IoT dan Webserver
